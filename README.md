🧮 React Calculator

A functional, responsive calculator application built as part of the Meta Front-End Developer Professional Certificate. This project focuses on managing application state and utilizing React Refs to handle direct DOM interactions.

✨ Features

Core Arithmetic: Performs addition, subtraction, multiplication, and division.
Result Persistence: Displays the current total and updates dynamically as operations are performed.
Input Management: Built using useRef to handle the input field and reset functionality.
Responsive UI: Optimized for both desktop and mobile viewing with clean, accessible styling.

🛠️ Tech Stack
Framework: React.js
Hooks Used: useState, useRef
Styling: CSS3 (custom layouts for a modern calculator feel)

📂 Project Highlights
State Control: Managed a single source of truth for the calculator's result.
DOM Manipulation: Leveraged the useRef hook to clear input fields and reset the application state efficiently.E
rror Handling: Implemented logic to handle edge cases in arithmetic operations.
